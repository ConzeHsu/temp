notes

  为了摆脱鼠标的依赖，提高编码效率，借助vimtutor学习了一下vim的基本操作，vimtutor的说明比较详细，在其中也提供了相应的练习和总结。

  直接在shell下键入vimtutor即可。

vimtutor

Lesson 1 Summary

  1. h (left)、j (down)、 k (up)、l (right)：光标方向控制

  2. vim filename：在shell上用命令行vim一个文件

  3. :q!：退出Vim，不保留任何修改

    :wq：保留修改

  4. x：删除当前光标的字符（vim命令模式下的 Backspace）

  5. i：插入字符:

    a：当前光标后追加字符

    A：当前行尾追加字符

Lesson 2 Summary

  1. dw：删除当前光标到当前单词尾（包括空格，通常把光标放到单词首以达到删除单词的效果）

  2. de：删除当前光标到单词尾，不包括空格

  3. d$orD：从当前光标删除到行尾

  4. dd：删除光标所在的整行（删除多行，eg：ndd）

  5. 2w：光标移动2个单词到词首（数量可变）

  6. 3e：光标移动到从当前位置后第三个单词的词尾

  7. 0：光标移动到行首

  8. 撤销前一步操作: u (lowercase u)
    撤销当前行的所有之前操作: U (capital U)
    撤销之前的撤销：CTRL-R

  9. operator [number] motion：

    eg： d2w

Lesson 3 Summary

  1. p：把之前的删除的内容追加到光标后面
  2. r：替换单个字符
  3. ce：改变从当前光标到单词结束的字符
  4. c$：改变从当前光标到行尾的字符
  5. c2：删除下面两行

c与d的区别：前者执行完之后直接进入insert模式，后者还是命令模式

Lesson 4 Summary

  1. CTRL-G：显示当前文件的路径，并显示当前光标在整个文件中的位置（%）

    G：移动到文件最后一行

    580+G：移动到文档的580行

    gg：移动到文件第一行

  2. /+key：向后搜索关键词，后面加关键词，n相同方向下一个，N反向上一个

    ?+key：向前搜索关键词

  3. %：来匹配相对应的 )，]，}、

  4. :s/old/new：在一行中用new来替换old

    :s/old/new/g：替换一行中所有的old，（g：global）

    :#,#s/old/new/g：在#到#两行间（行号，可以事先通过CTRL-G知道行号），以new 替换所有的old

    :%s/old/new/g：替换文件中的所有事件

    :%s/old/new/gc：每次替换前确认

Lesson 5 Summary

  1. :!：执行外部命令，例如:!ls显示当前目录下的文件

  2. :w filename：将当前vim文件写入到filename的文件中

  3. v motion :w filename：保存选中的的行到filename的文件中

  4. :r filename：把filename文件中的内容插入到当前光标的下一行

  5. :r !dir：把当前目录的文件信息插入到当前光标的下一行

Lesson 6 Summary

  1. o：小写，在光标下一行插入

    O：大写，在光标上一行插入

  2. a：在当前光标后插入

    A：在当前行尾插入

  3. e：光标移动到当前/下一个词尾

  4. r：当前字符单个替换

    R：从当前位置起覆盖替换

Lesson 7 Summary

  1. :helporF1orHelp：打开一个帮助窗口
  2. :help cmd：在cmd下打开一个帮助窗口
  3. CTRL-W：在不同窗口间切换
  4. :q：关闭帮助窗口

Lesson 8 Summary

  1. ctrl-forctrl-b：整页下翻/上翻（forword/backward）
  2. ctrl-dorctrl-u：翻半页（down/up）
  3. ctrl-eorctrl-y：滚一行
  4. zz：让光标所在的行居屏幕中央
  5. zt：让光标所在的行在屏幕最上一行（t=top）
  6. zb：让光标所在的行在屏幕最下一行（b=bottom）
